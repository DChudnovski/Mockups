# Notifications in 47th

## Categories

### Message

### Memos

Memos refer to items that are being requested by 47th for sales on the platform/showroom requests, B2B Marketplace Memos, Items that have had media service requests approved etc.

### Offers

Offers refer to offers on items that have MAO enabled. The offers come from the NXC website on the relevant listings.

### Analytics Reccs.

Where notifications would come in that can give recommendations based on the analytics status of items in a Vendor's inventory.

### Website Acc. Requests

This would notify about requests from the Website for accounts that would require review.

#### Settiings Features

We should allow for push on each of these types of notifications in the Manage section of 47th St./IG, should allow for them to send emails as well for further notification. There are some types that will always have emails associated with them (Namely memos and action items that involve 47th Management)
